Oppgave 2

$$ \forall n \ge 2, x_n = cos(x_{n - 1})sin(x_{n - 2}) $$ Base case, n = 2: $$ x_2 = cos(x_{n - 1})sin(x_{n - 2}) = cos(1.4)sin(2) ~= 0.15, $$ $$ 0 \le x_2 = 0.15 \le 1 $$ Induction case: Assume: $$ \forall k, 2 \le k \le n - 1, 0 \le x_k \le 1 $$ We need to know the following helper sentences (can be proved using IVT): $$ 0 \le x_k \le 1 \implies 0 \le cos(x_k) \le 1 $$ $$ 0 \le x_k \le 1 \implies 0 \le sin(x_k) \le 1 $$ $$ 0 \le a,b \le 1 \implies 0 \le a*b \le 1 $$ Now we want to prove for the case k = n: $$ k = n, x_k = cos(x_{k - 1})sin(x_{k - 2}) = ab, a = cos(x_{n - 1}), b = sin(x_{n - 2}) $$ Using induction hypothesis: $$ 0 \le x_{n - 1} \le 1 \wedge 0 \le x_{n - 2} \le 1 $$ Now using the three helper sentences the case is proved.
